### Workflow-demo repo

This is a repository to figure out how github workflows work, with the hope of creating an automated process for the wider team.

The idea is that when a feature branch is merged into a release branch, the feature branch will be deleted. And the hope is that we can use an automated github workflow for this.

This repository is created to explore workflows and try solidify the understanding of how they work, what they can do and how we can use them to complete the task above.
